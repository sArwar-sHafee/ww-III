# Component: Unit System

## Responsibility
Training units, tracking upkeep, and exposing attack/defence values for combat resolution.

## Rules
- Military and defence rosters are separate UI sections.
- Assault-capable units contribute their `attack` stat in the War Room.
- Defensive units contribute their `defense` stat when absorbing assaults.
- Missile units are trained like other units but are consumed when launched.
- Scout Drone remains a non-assault utility unit.
