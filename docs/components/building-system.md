# Component: Building System

## Responsibility
Queue support and economy structures, complete them over time, and apply capacity or production effects.

## Rules
- Build costs are paid immediately.
- Build completion increments the owned count.
- `Shelter` replaces `House` and adds population capacity.
- `Land Mine` replaces `Wall` as the basic ground-defence structure.
- `Missile Silo` is no longer a building; it is a research unlock.
